# Better Every Day

A responsive full-stack web application featuring a personal to-do list and a "Self-Talk" audio section.

## Features
- **To-Do List**: Manage daily tasks.
- **Self-Talk**: Record short audio clips, store them securely, and play them back.

## Tech Stack
- **Backend**: Node.js, Express, Prisma (PostgreSQL), Cloudflare R2
- **Frontend**: React, TypeScript, TailwindCSS, Vite
